# Ankamantatra

## Description
This is the first hackaton project in 42 Antananarivo.
Hackamantatra is a malagasy traditional guessing game.

## Installation
Clone the repository to your computer, Then
```bash
$cd Ankamantatra
$ make
$ ./hackamantatra
```
## Commande
```bash
>/afa-po            //To skip a question
>/s                 \\To skip a question
>/q                 /*To quit*/
>/quit              /*to quit*/
```
